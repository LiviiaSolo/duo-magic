# Duo Magic — Bubble & Balloon Show Website

Duo Magic is a vibrant promo website created for an artistic duo specializing in bubble shows, balloon twisting performances, neon acts, and musical entertainment.
The design was crafted by a professional designer in Figma, and the website was hand‑coded with a focus on responsiveness, smooth animations, and a bright, immersive show atmosphere.


## Features

- Responsive design
- Burger menu
- Bright visual style
- Photo gallery
- Smooth animations
- Contact forms
- Service presentation blocks


## Technologies

- HTML5
- CSS3 (Flexbox, Grid, animations)
- JavaScript
- Responsive Web Design
- GitHub Pages (deployment)
- Figma (design provided by designer)


## My Role

- Layout development — building all pages based on the provided Figma design.

- Responsive design — adapting the website for mobile devices, tablets, and large screens.

- Animations — creating smooth transitions and effects that enhance the festive atmosphere of the show.

- Image optimization — optimizing photos for fast loading without losing quality.

- UI adjustments — making small design refinements to improve readability and visual harmony.

- Deployment preparation — preparing the project structure for deployment on GitHub Pages.
